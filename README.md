# Platform-Shooter
A basic platform-shooter with Godot Engine. 

This project doesn’t have sprites. So, enable ‘Visible Collision Shapes’ in the Debug menu to see the collision boxes while running the game.


## Commit Logs

### First Commit

### Third Commit

### Fourth Commit
